
# MY Pinterest Clone 🎨

A modern, high-performance Pinterest UI clone built using **React** and **Tailwind CSS**. This project focuses on a clean user experience and real-time search functionality.

## ✨ Features
- **Working Search:** Get accurate photos based on your search query.
- **Dynamic Navigation:** Seamlessly switch between Home, Explore, News, About, and Business views.
- **Newsroom Section:** Includes a dedicated blog-style layout and a special CEO message section.
- **Fully Responsive:** Optimized for all screen sizes with a masonry-style image grid.
- **Interactive UI:** Includes login modals, hover effects, and smooth category filtering.

## 🚀 Tech Stack
- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **Icons:** Lucide-React / SVG
- **Images:** Live fetching via Image APIs
